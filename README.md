# libmedium
A safe rust library to communicate with the sysfs interface of lm-sensors.

## Getting Started

Just add this to your `Cargo.toml` file:

```
[dependencies]
libmedium = "0.2"
```

Look in the [examples](examples) folder for information on how to use libmedium.
